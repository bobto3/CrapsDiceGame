# CrapsDiceGame

## Craps! is a dice and card game inspired by the classic casino table game. Using a standard deck of cards, players flip one card at a time to set the point number, then roll three dice trying to hit it.

## Game Rules:
## Players take turns rolling all three dice, aiming to hit the point number on the face-up card. Any combination of dice is valid (i.e., a point number can be scored using just one die or by adding the values of two or three dice). All three dice are rolled every turn.

## When the number on the card is hit, the roller wins the card and scores 1 point. Flip over a new card to establish a new point number.

## If a roller scores a point or hits a hardway, they keep the dice and roll again.
## The first player to 7 points wins the round and craps out!

## Point Numbers:
## Cards 2–10: Use the number on the card
## Face cards: Each roller picks a number between 10–18
## Ace: The first roller picks 1 or 11
## Joker: The first roller picks any number
##
## Hardways (Special Rolls):
## Any three of a kind
## Any straight
## Bonus Points (Special Rolls Worth 1 Point):
## All 1s or all 6s
## A straight or three of a kind while hitting the point number (the roller scores 2 points)
